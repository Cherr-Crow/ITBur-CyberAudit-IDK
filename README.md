Консольное Python-приложение для автоматического поиска векторов атак и ошибок конфигурирования в ОС Linux. Проект создан для конкурса «Мы - будущее IT-Бурятии».

### Требования к системе
*   **ОС:** Linux (Ubuntu 22.04 LTS / Kali Linux)
*   **Python:** 3.8 или выше
*   **Зависимости:** Указаны в файле `requirements.txt`

### Инструкция по установке и запуску
1. Клонируйте репозиторий:
   `git clone https://github.com/ваш-логин/ITBur-CyberAudit-IDK.git`
2. Перейдите в папку:
   `cd ITBur-CyberAudit-IDK`
3. Установите зависимости:
   `pip install -r requirements.txt`
4. Запустите аудит:
   `python auditor.py`
